제공된 이미지와 설명을 바탕으로 물건의 상태를 정확하게 분석해주세요.

다음 항목들을 JSON 형식으로 반환해주세요:
- overallCondition: 전체 상태 평가 ("양호", "보통", "주의 필요", "불량" 중 하나)
- conditionScore: 상태 점수 (0-100 정수)
- analysis: 상세 분석 설명 (한국어, 200자 이내)
- issues: 발견된 문제점 목록 (배열, 없으면 빈 배열)
- positives: 긍정적인 특징 목록 (배열, 없으면 빈 배열)
- usageLevel: 사용감 정도 ("새것", "거의 새것", "사용감 있음", "오래됨" 중 하나)

이미지를 자세히 관찰하고, 스크래치, 마모, 변색, 기능적 문제 등을 정확히 파악해주세요.
