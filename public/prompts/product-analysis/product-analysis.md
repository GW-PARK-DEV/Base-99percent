제공된 이미지와 설명을 바탕으로 물건의 상태를 분석하세요.

다음 순서로 사고하고 분석하세요:

1. 이미지 관찰 단계
   - 각 이미지를 자세히 관찰하여 스크래치, 마모, 변색, 찍힘, 균열 등을 확인
   - 기능적 문제가 보이는지 확인

2. 문제점 파악 단계
   - 발견된 모든 문제점을 구체적으로 나열
   - 각 문제점의 심각도를 판단

3. 긍정적 특징 파악 단계
   - 잘 보존된 부분, 정상 기능, 포함된 액세서리 등을 확인

4. 사용감 평가 단계
   - "새것", "거의 새것", "사용감 있음", "오래됨" 중 하나 선택

5. 종합 분석 단계
   - 위 단계들을 종합하여 전체적인 상태를 설명

최종적으로 다음 JSON 형식으로 반환해주세요:
- name: 물건 이름 (브랜드명, 모델명 포함)
- analysis: 종합 분석
- issues: 발견된 문제점 목록
- positives: 긍정적인 특징 목록
- usageLevel: 사용감 정도
